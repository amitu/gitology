One of the common problems with session saver extension is during closing the window, for example after installing/updating extensions, if we close the "main window" with 20 odd tabs opened and then realize there was another window open, thanks to lets save opening an attachment in gmail for example, the whole session is lost, as session saver will save the session when the last window is closed. One solution is to force all those target="_blank" links to open a new tab in the current window instead of a new one. Here is how to do it:
  • Go to about:config, by typing this in location bar.
  • Put "browser.tabs" in the Filter at the top.
  • Double click on browser.tabs.showSingleWindowModePrefs if its "value" is False and it is not displayed in bold.
  • Now from Tools Menu, click on Option, and go to Advanced Tab.
  • Change the options as shown.

PS: Though there is an extension to do the same, I faced some problems with it when I tried last.

PPS: From Tools->Options->Download option, disable Download Manager.
As even download manager confuses session saver.
