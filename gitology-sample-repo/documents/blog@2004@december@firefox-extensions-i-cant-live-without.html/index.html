Must haves:

  • Focus Last Selected Tab: instead of the last open tab. Absolute must have.
  • Session Saver: Remembers all opened tabs when closing browser, and reloads them at startup.
  • Mouse Gestures: Allows you to execute common commands (like page forward/backward, close tab, new tab) by mouse gestures drawn over the current webpage, without reaching for the toolbar or the keyboard.
  • Add Bookmark here: Must have if you still believe in the concept of bookmarks and rely on them on day to day life, lets you directly add the current page to the folder you chose.
  • Nuke Anything: Lets you remove anything from the current page, imagine having to discuss a page with people, that has some inappropriate image/something; highlight it, right click on it, remove it. And ofcourse its very fun too.
  • CuteMenus: Without this extension all the menus look terribly bald, adds icons next to menu labels.

Also Usefuls:

  • Resize Search Box: Minor Utility extension, lets you resize the built-in google search input field.
  • AutoCopy: For the poor fellows who are stuck with windows, and don't see the need of pressing Ctrl-C after selecting to put text into clipboard.
  • Reload Every: Reloads the page every so often. Esp. useful with sports live score pages.
  • BlogThis: Convenient if you are a blogger.
  • User Agent Switcher: Adds a menu to switch the user agent of the browser.
  • Image Show Hide: Shift-B and images are gone!
  • WebmailCompose: Mailto-s open in GMail's Composer.
  • Tabbrowser Preferences: Install it and you will know.
  • Stop/Reload: Provides a space-saving combined Stop and Reload toolbar button.
