 जय हिन्द
If you can not view it properly try setting the encoding (Menu->View->Character Encoding) to Unicode, and if even that doesn't help, stop placing too much faith in your luck. Anyways here is how to compose in hindi, get the unicode editor, yudit, from here(fonts). Download raghu.ttf to "C:\Program Files\Yudit\fonts" folder on windows, and run the program.


In Yudit you will notice a single line input field at the bottom of the window, enter "test" in it and press "Return", it will show various languages rendered in their native fonts, look for Hindi and make sure you can see stuff displayed properly, as in the image above.

Congratulate yourself if so. Rest is easy, click on the Keymap selection button, and select Devnagri-inscript and Devnagri-Phonetic. In yudit, you can quickly switch the keymap by assigning keymaps to different "Function Keys" F1 F2 etc.



Select a Function key in the middle, select a keymap from the left pane, and press the blue triangle button to assign. Now you will need to learn the keymaps, phonetic is easy, n is for न, d is for द and so on, but I get stuck on मात्रा, so I use inscript too, its keymap can be found here.

Now you should be able compose in hindi, just copy paste in the blogger and you are done. One minor thing, go to Setting->Formatting page on blogger and make sure the Encoding is Universal(Unicode UTF-8).
