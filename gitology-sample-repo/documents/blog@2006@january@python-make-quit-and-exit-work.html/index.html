Hans Nowak writes about how python interactive shell should quit/exit
when you type 'quit' or 'exit'.
He also shows an example.
>>> class _Exit:
... def __repr__(self):
... raise SystemExit
...
>>> exit = _Exit()
>>> exit # exits the interpreter
