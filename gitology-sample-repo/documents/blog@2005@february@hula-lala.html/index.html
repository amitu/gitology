Hula project has been launched, its an open source mail and calendar server that "people would love to use".

We have already have tons of open source groupware, so why is this important? First of all, its based on a commercial tool (NetMail) by Novell, which supposedly already runs millions of mailboxes and calenders. Secondly their projects main aim is to be cool and a product that people would love to use. They are trying to make a rich DHTML GMail like webmail interface for example. More on Nat's blog. Lets see how well this one goes.
